NERD SNAKE

Nerd Snake is a ludic solution to learn the basics of HTML and CSS. Each player will answer technical and general questions. When a player gets right, he receives a shortcode card as a point. After all the players reach the end of the path, they will need to combine all the cards to make a web page.
